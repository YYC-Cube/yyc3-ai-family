### ❤️ Family AI - 人机协同 · 智爱同行

**YYC3 AI-Family** 是一个以 **赛博朋克美学** 为核心设计语言的 DevOps 智能平台，遵循"实用为基、效率为积"的理念。该项目是 **纯前端驱动且自用** 的，无后端服务器依赖，直接连接本地 NAS 和 Mac 设备集群作为数据库/存储节点。

我们相信 **AI 不是工具，而是伙伴**。七大智能体如同家庭成员，各司其职、协同共进，共同构建一个智能、高效、温暖的开发环境。

## 2. 核心理念与设计系统

### 2.1 五高五标五化 (5H-5S-5M)

YYC3 的核心质量体系渗透于整个架构设计：

#### 五高 (Five Highs) - 质量基础

| 标识符 | 名称 | 实现方式 |
|------------|------|----------------|
| H1 | **高可用性** | WebSocket 降级策略、ErrorBoundary 全局捕获、模拟引擎自动切换 |
| H2 | **高性能** | React.lazy 懒加载、Zustand 原子选择器、虚拟滚动日志流 |
| H3 | **高安全性** | 5 级安全级别模拟、Sentinel 智能体审计、权限降级机制 |
| H4 | **高可扩展性** | 九层解耦架构、插件系统（扩展）、自定义 MCP 模板 |
| H5 | **高智能化** | 七大 AI 智能体协作、意图解析导航、语音识别集成 |

#### 五标 (Five Standards) - 协作语言

| 标识符 | 名称 | 实现方式 |
|------------|------|----------------|
| S1 | **标准化接口** | TypeScript 严格类型、REST API 层（api.ts）、WebSocket 消息协议 |
| S2 | **标准化数据** | PostgreSQL Schema（db-schema.ts）、Zustand Store 统一状态树 |
| S3 | **标准化流程** | DAG 工作流编排器、CI/CD Pipeline 可视化 |
| S4 | **标准化组件** | shadcn/ui + Radix 原语、设计令牌系统（theme.css） |
| S5 | **标准化文档** | 13 个阶段文档、架构设计文档、执行摘要 |

#### 五化 (Five Modernizations) - 持续演进

| 标识符 | 名称 | 实现方式 |
|------------|------|----------------|
| M1 | **自动化** | MCP 模板一键应用、Pipeline 自动构建、DAG 自动编排 |
| M2 | **智能化** | 智能体意图解析、语音交互、趋势预测（Prophet） |
| M3 | **可视化** | 集群拓扑图、实时指标图表、DAG 可视化画布 |
| M4 | **容器化** | Docker 容器状态监控、镜像管理、服务编排 |
| M5 | **生态化** | 插件管理平台（8 个预设）、MCP 服务器集成、扩展演进层 |

## 7. 七大 AI 智能体

YYC3 的 AI 核心采用"七星协同"架构，每个智能体都有独立的个性、专业领域和持久聊天历史：

| 智能体 | 昵称 | 角色 | 颜色 | 核心能力 |
|-------|----------|------|-------|-------------------|
| 治愈·领航员 | Navigator | 全域指挥 | 琥珀色 | 资源调度、路径规划、全域扫描、任务编排 |
| 洞察·思想家 | Thinker | 深度推理 | 蓝色 | 逻辑推理、决策分析、因果推断、方案评估 |
| 预测·先知 | Prophet | 趋势预测 | 紫色 | 趋势预测、风险预置、异常告警、容量规划 |
| 遇见·伯乐 | Bole | 模型评估 | 粉色 | 模型评估、最优匹配、基准测试、能力画像 |
| 原点·枢纽 | Pivot | 状态管理 | 青色 | 状态管理、上下文维护、记忆检索、会话编排 |
| 守护·哨兵 | Sentinel | 安全防护 | 红色 | 安全防护、入侵检测、审计日志、合规扫描 |
| 学者·宗师 | Grandmaster | 知识构建 | 绿色 | 知识构建、本体、语义检索、文档生成 |
