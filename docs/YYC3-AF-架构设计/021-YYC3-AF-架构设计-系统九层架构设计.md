## 🏗️ 九层功能架构体系设计

### 架构设计原则

基于"五高五标五化"核心机制，设计九层功能架构体系（八层架构 + 系统设置层），确保：

1. **职责边界清晰**：每层有明确的职责和边界，避免职责重叠
2. **协作关系明确**：层与层之间有明确的协作关系和接口定义
3. **高内聚低耦合**：层内功能高内聚，层间依赖低耦合
4. **可独立演进**：每层可以独立演进和升级，不影响其他层
5. **可水平扩展**：每层可以水平扩展，支持系统整体扩展

### 九层架构职责边界与协作关系

#### Layer-01 基础设施层（Infrastructure Layer）

**职责边界**：

- 提供物理硬件资源（服务器、存储、网络）
- 提供容器化部署环境（Docker、Kubernetes）
- 提供网络基础设施（负载均衡、CDN、专线）
- 提供安全基础设施（防火墙、IDS、IPS）
- 提供监控基础设施（Prometheus、Grafana）

**协作关系**：

- 向上提供：硬件资源、容器环境、网络连接、安全防护、监控数据
- 向下依赖：物理设备、网络运营商、云服务商
- 横向协作：与Layer-02协作提供数据存储资源，与Layer-03协作提供服务部署环境

**核心能力**：

- 硬件资源管理
- 容器编排
- 网络管理
- 安全防护
- 监控告警

**协作关系**：

- 向上提供：数据存储、数据查询、数据缓存、数据搜索、文件存储
- 向下依赖：Layer-01提供的存储资源和网络资源

## 🎨 Figma设计UI/UX提示词

### 设计原则与理念

**核心设计理念**：以"五高五标五化"为核心机制，打造智能化、标准化、可视化的用户界面

**设计关键词**：

- 智能化（Intelligent）：AI驱动、自适应、智能推荐
- 标准化（Standardized）：统一规范、一致性、可预测性
- 可视化（Visualized）：数据可视化、状态可视化、流程可视化
- 高效性（Efficient）：快速响应、简洁操作、流程优化
- 可扩展性（Scalable）：模块化设计、灵活配置、易于扩展

---

### 1. 全局设计系统（Design System）

#### 1.1 色彩系统（Color System）

**提示词**：

```
创建一个现代化的色彩系统，体现YYC³品牌特色：
- 主色调：使用深蓝色系（#1a365d）作为品牌主色，象征专业和可靠
- 辅助色：使用科技蓝（#3182ce）作为交互色，使用绿色（#38a169）作为成功状态，使用红色（#e53e3e）作为错误状态
- 中性色：使用灰色系（#718096、#a0aec0、#cbd5e0）作为文字和边框颜色
- 主题色：支持亮色主题和暗色主题切换，使用CSS变量实现动态主题切换
- 语义色：为不同状态和功能定义语义化颜色，确保视觉一致性
- 无障碍：确保色彩对比度符合WCAG 2.1 AA标准，支持色盲用户
```

#### 1.2 字体系统（Typography System）

**提示词**：

```
设计一个清晰的字体系统，确保可读性和层次感：
- 字体家族：使用系统字体栈（San Francisco、Segoe UI、Roboto等）确保跨平台一致性
- 字体大小：建立12px到32px的字体大小层级，使用rem单位实现响应式缩放
- 字重：使用400（Regular）、500（Medium）、600（SemiBold）、700（Bold）四个字重
- 行高：根据字体大小设置合理的行高（1.4-1.6），确保良好的阅读体验
- 字间距：根据字体大小和语言调整字间距，中文使用0.05em，英文使用0.01em
- 标题层级：建立H1-H6的标题层级，每个层级有明确的样式和用途
```

#### 1.3 间距系统（Spacing System）

**提示词**：

```
建立一个统一的间距系统，使用4px基准网格：
- 基准单位：使用4px作为基准单位，间距值为4、8、12、16、20、24、32、40、48、64、80、96
- 组件间距：组件内部使用8-16px间距，组件之间使用16-32px间距
- 页面间距：页面边距使用24-48px，内容区块之间使用32-48px间距
- 响应式间距：在小屏幕上减少间距，在大屏幕上增加间距
- 一致性：所有间距使用预定义的值，避免随意使用
```

#### 1.4 圆角系统（Border Radius System）

**提示词**：

```
设计一个现代的圆角系统，平衡美观和功能性：
- 圆角值：使用4、8、12、16、24px作为圆角值
- 小圆角（4px）：用于按钮、输入框、标签等小元素
- 中圆角（8px）：用于卡片、模态框等中等元素
- 大圆角（12-16px）：用于对话框、提示框等大元素
- 超大圆角（24px）：用于特殊容器、图片等超大元素
- 一致性：相同类型的元素使用相同的圆角值
```

#### 1.5 阴影系统（Shadow System）

**提示词**：

```
创建一个层次分明的阴影系统，增强视觉深度：
- 阴影层级：建立4-6个阴影层级，从轻微到明显
- 轻微阴影：用于悬停状态、轻微浮起元素
- 中等阴影：用于卡片、模态框等常规浮起元素
- 明显阴影：用于下拉菜单、弹出框等需要强调的元素
- 阴影颜色：使用半透明的黑色（rgba(0,0,0,0.1-0.3)）作为阴影颜色
- 阴影模糊：使用4-32px的模糊半径，根据阴影层级调整
- 暗色主题：在暗色主题中使用浅色阴影或发光效果
```
