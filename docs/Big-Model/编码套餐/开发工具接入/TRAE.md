# TRAE

> 在 TRAE CN 中使用 GLM Coding Plan 的方法

TRAE 是一款面向开发者打造的智能 IDE，融合 AI 问答、行内代码补全及 Agentic Coding Workflow 等功能为一体，构建了新一代的编程工作流。

<Tip>
  搭配 [**GLM Coding Plan**](https://zhipuaishengchan.datasink.sensorsdata.cn/t/Nd)，TRAE 能显著提升开发效率，让代码编写更高效、调试更智能，并自动化处理各类重复性任务。
</Tip>

<Tip>
  使用 GLM-5，需要选择对应的模型编码为 `GLM-5`
</Tip>

## 一、安装 TRAE CN

访问 [TRAE](www.trae.cn) 官网下载并安装适合您的操作系统的版本。

![Description](https://cdn.bigmodel.cn/markdown/1768878902816image.png?attname=image.png)

## 二、配置 GLM Coding Plan

### 1. 添加自定义模型

在 TRAE 中，点击**模型切换下拉菜单**或**进入 Settings/Models**，选择添加模型

![Description](https://cdn.bigmodel.cn/markdown/1768879303571image.png?attname=image.png)

### 2. 在模型服务商中选择 Bigmodel-Plan

![Description](https://cdn.bigmodel.cn/markdown/1768879798001image.png?attname=image.png)

### 3. 选择使用的模型

选择 `GLM-4.7`

![Description](https://cdn.bigmodel.cn/markdown/1768879769159image.png?attname=image.png)

### 4. 填写你的 API Key

添加 API Key，点击 Get API Key 可跳转至智谱平台获取你的API key

![Description](https://cdn.bigmodel.cn/markdown/1768879742082image.png?attname=image.png)

## 三、保存并切换模型

配置完成后，保存设置并在 **自定义模型** 中选择您刚创建的 **`GLM-4.7`**。

![Description](https://cdn.bigmodel.cn/markdown/1768879957907image.png?attname=image.png)

<Tip>
  完毕！现在你就可以在 TRAE 中使用 **`GLM-4.7`** 模型进行代码生成、调试、任务分析等工作了。
</Tip>
