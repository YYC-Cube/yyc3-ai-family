# Cherry Studio

> 在 Cherry Studio 中使用 GLM Coding Plan 的方法

将 **GLM-4.7** 模型通过 OpenAI 协议在 **Cherry Studio** 中自定义配置模型接入使用。

### 1. 安装 Cherry Studio

访问 [Cherry Studio](https://www.cherry-ai.com/) 官网下载并安装适合您的操作系统的版本。

### 2. 配置 Api Key

在 Cherry Studio 中，打开 “**设置->模型服务**” 部分。

![Description](https://cdn.bigmodel.cn/markdown/1768528979029image.png?attname=image.png)

* 选择 **智谱开放平台**。
* 配置 **API Key**（从[智谱开放平台](https://bigmodel.cn/usercenter/proj-mgmt/apikeys)获取）。
* 变更 API 地址 为 `https://open.bigmodel.cn/api/coding/paas/v4/`
* 点击下方管理，将 `GLM-4.7` 模型添加。

![Description](https://cdn.bigmodel.cn/markdown/1768529184200image.png?attname=image.png)

### 3. 开始使用

配置完成后，在对话中选择您刚配置的 **GLM-4.7** 使用。

![Description](https://cdn.bigmodel.cn/markdown/1768529370867image.png?attname=image.png)
