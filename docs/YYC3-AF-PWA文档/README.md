## 文档规范

### 文档命名规范

**格式：** `{编号}-YYC3-AF-PWA-{文档类型}.md`

**示例：**
- 241-YYC3-AF-PWA-设计文档.md
- 242-YYC3-AF-PWA-实现文档.md
- 243-YYC3-AF-PWA-测试文档.md

### 文档头部规范

**必需字段：**
- `@file`: 文件名
- `@description`: 文档描述
- `@author`: 作者
- `@version`: 版本号
- `@created`: 创建日期
- `@updated`: 最后更新日期
- `@status`: 状态（draft、published、deprecated）
- `@tags`: 标签列表

**示例：**

```markdown
---
@file: 242-YYC3-AF-PWA-实现文档.md
@description: YYC3-AF-PWA实现文档，包括PWA配置、Service Worker、Manifest、测试验证等。
@author: YanYuCloudCube Team
@version: v1.0.0
@created: 2026-02-18
@updated: 2026-02-18
@status: published
@tags: [PWA],[实现文档],[Service Worker],[Manifest],[测试验证]
---
```

### 文档内容规范

**结构要求：**
1. 概述（Overview）
2. 核心内容（Core Content）
3. 附录（Appendix）

**内容要求：**
- 使用 Markdown 格式
- 代码块使用语法高亮
- 表格使用 Markdown 表格格式
- 图片使用相对路径
- 链接使用绝对路径或相对路径
