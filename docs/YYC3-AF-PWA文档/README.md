## PWA特性

### 核心功能

- **离线访问**: Service Worker缓存支持
- **桌面安装**: 添加到主屏幕功能
- **推送通知**: Web Push API集成
- **自动更新**: 后台自动更新机制
- **响应式设计**: 适配各种设备屏幕

### 技术栈

- **框架**: React + TypeScript
- **构建工具**: Vite
- **PWA库**: vite-plugin-pwa
- **Service Worker**: Workbox
- **Manifest**: Web App Manifest

### 浏览器支持

- Chrome: ✅ 完全支持
- Firefox: ✅ 完全支持
- Safari: ⚠️ 部分支持
- Edge: ✅ 完全支持
