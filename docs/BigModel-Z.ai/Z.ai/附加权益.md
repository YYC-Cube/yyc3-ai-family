## 智谱 AI 输入法

### 产品简介

智谱 AI 输入法是一款基于先进 AI 技术的智能语音输入助手，通过简单的 Fn 快捷键操作，即可将您的语音实时转换为高质量文字。产品可在微信、飞书、Word、代码编辑器等各类输入框中使用，无论是日常聊天、工作文档还是专业术语，都能准确识别并转换，大幅提升您的输入效率。

**现已同时支持 macOS 与 Windows 系统**

### 专属权益

- **面向公众永久免费使用**
- **GLM Coding Plan 用户每月首次使用可获得 1000 积分**
- 积分可用于 [AutoGLM 网页端](https://autoglm.zhipuai.cn/) 视频总结等高阶功能

### 核心功能

#### 精准识别

- 错误率极低
- 专属词汇识别
- 中英混合识别
- 行业术语识别表现优异

#### 内置多重人设

- 在菜单栏一键切换不同内置人设
- 适应不同对话场景
- 支持自定义人设

#### 语音召唤 AI

- 选中文本或轻呼"小凹"
- 即可唤醒 GLM 模型
- 通过语音直接下达复杂任务指令

### 使用方法

#### 1. 安装与设置

1. 访问 [智谱 AI 输入法官网](https://autoglm.zhipuai.cn/autotyper/) 下载并安装应用
2. 使用与 GLM Coding Plan 套餐相同手机号登录
3. 完成新手引导（新手引导很重要！）
4. 设置偏好人设和语音输入快捷键
5. 设置专属热词：在设置菜单中找到"词典"，添加您常用的专属名词

#### 2. 基本操作

##### 语音输入

1. 按下 Fn 键（或自定义快捷键）
2. 开始说话
3. 释放按键，自动转换文字

##### 召唤 AI

1. 选中文本
2. 轻呼"小凹"
3. 下达语音指令
4. AI 自动执行任务

##### 切换人设

1. 点击菜单栏图标
2. 选择"人设"
3. 选择适合当前场景的人设

## AutoGLM OpenClaw

### 简介

AutoGLM OpenClaw 是 GLM Coding Plan 的移动端扩展，让您在手机上也能享受智能编码体验。

### 功能特性

- **跨设备同步**：与桌面端配置同步
- **语音编码**：语音输入代码需求
- **移动端访问**：随时随地使用 GLM 能力
- **智能补全**：移动端代码补全建议

### 使用方法

1. 下载 AutoGLM App
2. 使用 GLM Coding Plan 账号登录
3. 选择 OpenClaw 模式
4. 开始使用

### 适用场景

- 紧急修复：在移动端快速解决简单问题
- 代码预览：查看和讨论代码
- 文档编写：在移动端编写文档
- 团队协作：参与代码讨论
