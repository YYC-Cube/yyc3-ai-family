## 核心能力

### 编码工具支持

Claude Code、Cline、Cursor、Roo Code、Kilo Code、OpenCode、OpenClaw、Crush、Goose、TRAE 等 15+ 主流编程工具

### MCP 服务器

视觉理解、联网搜索、网页读取、开源仓库访问四大能力

### 专属模型

GLM-5、GLM-4.7、GLM-4.5-Air 等编码优化模型
