# 快速开始

> 5 分钟上手 GLM Coding Plan，在您的编程工具中享受智能编码体验。

## 前提条件

- 已订阅 GLM Coding Plan (Lite/Pro/Max)
- 已获取 API Key
- 拥有支持的编程工具

## 步骤一：订阅套餐

1. 访问 [GLM Coding Plan 订阅页面](https://zhipuaishengchan.datasink.sensorsdata.cn/t/Nd)
2. 选择适合的套餐：
   - **Lite 套餐**：适合个人开发者，每 5 小时约 80 次调用
   - **Pro 套餐**：适合专业开发，每 5 小时约 400 次调用
   - **Max 套餐**：适合团队使用，每 5 小时约 1600 次调用
3. 完成支付

<Tip>
  当前 Max 与 Pro 套餐均已支持 GLM-5，后续待新老模型资源迭代完成，Lite 也将支持 GLM-5。
  GLM-5 参数量更大，对标 Claude Opus 模型，调用 GLM-5 时用量将按照 "高峰期 3 倍，非高峰期 2 倍" 进行计算抵扣。
  高峰期为每日的 14:00～18:00（UTC+8）。
</Tip>

## 步骤二：获取 API Key

1. 登录 [智谱开放平台](https://open.bigmodel.cn)
2. 进入 [API Keys 页面](https://bigmodel.cn/usercenter/proj-mgmt/apikeys)
3. 创建新的 API Key

<Warning>
  请妥善保管 API Key，不要泄露给他人。Coding Plan Key 就是普通 API Key，只是使用场景不同。
</Warning>

## 步骤三：选择开发工具

根据您的开发习惯选择合适的工具：

### 终端开发
- **Claude Code** - 终端智能编码助手
- **OpenCode** - 终端 + IDE 编程代理
- **Crush** - CLI + TUI 编程工具
- **OpenClaw** - 本地 AI 助手

### VS Code 插件
- **Cline** - VS Code AI 插件
- **Roo Code** - VS Code 项目分析助手
- **Kilo Code** - VS Code 代码生成插件
- **Cursor** - AI 驱动的代码编辑器

### 其他工具
- **Goose** - AI Agent 工具
- **TRAE** - Trae IDE 集成
- **Cherry Studio** - 代码生成 IDE
- **Factory-Droid** - Android 开发工具

## 验证配置

配置完成后，在工具中测试：

1. 发送简单对话："你好"
2. 测试代码生成："帮我写一个 Python 函数"
3. 测试 MCP 功能（如已配置）

## 额度说明

- 每 5 小时刷新一次额度
- Lite 套餐：每月约 400 次调用
- Pro 套餐：每月约 2000 次调用
- Max 套餐：每月约 8000 次调用

查看详细额度：https://www.bigmodel.cn/usercenter/glm-coding/usage
