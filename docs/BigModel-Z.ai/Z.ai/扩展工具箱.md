## 一键安装小助手

### 简介

Coding Tool Helper 是一个编码工具助手，快速将您的 GLM 编码套餐加载到您喜爱的编码工具中。安装并运行它，按照界面提示操作即可自动完成工具安装、套餐配置、MCP 服务器管理等操作。

### 支持的工具

- Claude Code
- OpenCode
- Crush
- Factory Droid

### 快速开始

```bash
npx @z_ai/coding-helper
```

### 功能特性

- 交互式向导：友好的设置引导
- 套餐集成：自动配置 Coding Plan
- MCP 管理：一键启用/禁用 MCP 服务器
- 插件市场：Claude Code 插件管理
- 多工具管理：统一管理多个编码工具

### 使用步骤

1. 运行 `npx @z_ai/coding-helper`
2. 选择要配置的编码工具
3. 输入 API Key
4. 选择要启用的 MCP 服务器
5. 完成配置

## 用量查询插件

### 简介

**glm-plan-usage** 插件允许您在 Claude Code 中直接查询 GLM Coding Plan 的当前配额和使用统计信息。

### GitHub 仓库

[https://github.com/zai-org/zai-coding-plugins](https://github.com/zai-org/zai-coding-plugins)

### 前置要求

- Node.js 18 或更高版本
- 已安装 Claude Code CLI

### 安装方式

#### 方式一：手动安装

```bash
# 1. 安装插件市场
claude plugin marketplace add zai-org/zai-coding-plugins

# 2. 安装插件
claude plugin install glm-plan-usage@zai-coding-plugins
```

#### 方式二：使用一键安装小助手

```bash
npx @z_ai/coding-helper
```

选择：开始 -> 编码工具 -> Claude Code -> 插件市场

### 使用方法

在 Claude Code 中输入：

```
用量查询
```

或

```
quota
```

### 显示信息

- 当前套餐类型
- 每 5 小时剩余额度
- 每周剩余额度
- 已使用次数
- 刷新时间

## 问题反馈插件

### 简介

**glm-feedback** 插件允许您在 Claude Code 中快速提交使用问题和建议。

### GitHub 仓库

[https://github.com/zai-org/zai-coding-plugins](https://github.com/zai-org/zai-coding-plugins)

### 前置要求

- Node.js 18 或更高版本
- 已安装 Claude Code CLI

### 安装方式

```bash
# 1. 安装插件市场
claude plugin marketplace add zai-org/zai-coding-plugins

# 2. 安装插件
claude plugin install glm-feedback@zai-coding-plugins
```

### 使用方法

在 Claude Code 中输入：

```
反馈
```

或

```
feedback
```

### 反馈类型

- Bug 报告
- 功能建议
- 使用问题
- 性能问题

### 反馈内容建议

1. **问题描述**：详细描述遇到的问题
2. **复现步骤**：如何重现问题
3. **预期行为**：期望的正确结果
4. **实际行为**：实际发生的情况
5. **环境信息**：操作系统、工具版本等

## GLM-in-Excel

### 简介

GLM-in-Excel 是一款将 GLM 模型能力集成到 Excel 中的插件，让您在 Excel 中直接使用 AI 能力进行数据处理、公式生成、文本分析等。

### 功能特性

- **公式生成**：用自然语言描述需求，自动生成 Excel 公式
- **数据分析**：智能分析数据，生成可视化建议
- **文本处理**：文本提取、翻译、摘要等功能
- **智能补全**：智能补全表格内容

### 安装方法

1. 访问智谱开放平台下载插件
2. 在 Excel 中安装插件
3. 使用 GLM Coding Plan 账号登录

### 使用示例

#### 生成公式

```
用户：帮我计算 A 列的平均值
AI：=AVERAGE(A:A)
```

#### 数据分析

```
用户：分析这个销售数据表
AI：根据数据分析...
```

#### 文本处理

```
用户：提取 A 列中的邮箱地址
AI：=IF(ISNUMBER(SEARCH("@", A1)), A1, "")
```

## 常见问题

A: 当前支持 Claude Code、OpenCode、Crush、Factory Droid，持续扩展中。

### Q: 用量查询插件如何更新？

A: 运行 `claude plugin update glm-plan-usage@zai-coding-plugins`

### Q: 反馈后多久能收到回复？

A: 通常 1-3 个工作日内会有回复。

### Q: GLM-in-Excel 需要额外付费吗？

A: GLM Coding Plan 用户可免费使用，调用占用套餐额度。
