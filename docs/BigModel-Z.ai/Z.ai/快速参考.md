## 模型选择

| 模型 | 适用场景 | 特点 |
|-----|---------|------|
| **GLM-5** | 复杂架构、高级算法 | 智能最高，用量 2-3 倍 |
| **GLM-4.7** | 通用编码、日常开发 | 平衡性能与成本 |
| **GLM-4.5-Air** | 简单任务、快速响应 | 成本最低，响应最快 |

## 环境变量

### Claude Code

```bash
export ANTHROPIC_AUTH_TOKEN="your_api_key"
export ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/anthropic"
```

### Windows PowerShell

```powershell
$env:ANTHROPIC_AUTH_TOKEN = "your_api_key"
$env:ANTHROPIC_BASE_URL = "https://open.bigmodel.cn/api/anthropic"
```

## 常用命令

### Claude Code

```bash
# 安装
npm install -g @anthropic-ai/claude-code

# 查看版本
claude --version

# 运行
claude
```

### 检查配置

```bash
# 检查环境变量
echo $ANTHROPIC_AUTH_TOKEN
echo $ANTHROPIC_BASE_URL

# 检查配置文件
cat ~/.claude/settings.json
```

## 错误码速查

| 错误码 | 含义 | 解决方案 |
|-------|------|---------|
| **401** | 认证失败 | 检查 API Key |
| **403** | 权限不足 | 检查订阅状态 |
| **429** | 速率限制 | 降低请求频率 |
| **500** | 服务器错误 | 稍后重试 |

## 使用建议

### 模型选择

- **简单任务** → GLM-4.5-Air
- **通用编码** → GLM-4.7
- **复杂任务** → GLM-5

### 时间选择

- **非高峰期**（避开 14:00-18:00）使用 GLM-5
- **高峰期**使用 GLM-4.7

### MCP 使用
