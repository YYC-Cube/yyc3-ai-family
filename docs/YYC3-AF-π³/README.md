## 文档规范

### 命名规范

- 格式：`{编号}-{项目}-{模块}-{描述}.md`
- 编号：三位数字，按创建顺序递增
- 项目：YYC3-AF（YYC³ AI-Family）
- 模块：原型交互、设计系统等
- 描述：简短描述文档内容

### 文档头部

每个文档应包含以下头部信息：

```markdown
# 文档标题

> **文档编号**: {编号}
> **文档类型**: {类型}
> **创建日期**: {日期}
> **状态**: {状态}

---
```

### 文档内容

- 使用 Markdown 格式
- 代码块使用语法高亮
- 表格用于结构化数据
- 图片使用相对路径
- 链接使用相对路径

1. 确定文档编号（按顺序递增）
